# Extensions

The following the figure extension:

## Figures
 
A figure can be defined by using a pattern equivalent to a fenced code block but with the character `^`

```````````````````````````````` example
^^^
This is a figure
^^^ This is a *caption*
.
<figure>
<p>This is a figure</p>
<figcaption>This is a <em>caption</em></figcaption>
</figure>
````````````````````````````````
